# PaperTok 前端变更记录

## 2025-01-22

### 新增功能
- **完整的前端应用架构**
  - React 19 + TypeScript + Vite
  - React Router 路由管理
  - Context API 全局状态管理
  - Axios HTTP 客户端

### 核心组件
- **PaperCard** - 论文卡片组件
  - 封面图展示（支持降级到分类渐变占位图）
  - 标题、作者、摘要显示
  - 摘要展开/折叠功能
  - 点赞、收藏、分享、查看原文操作

- **CategoryFilter** - 分类筛选器
  - 快速切换论文分类
  - 支持常见 AI/CS 分类

- **LoadingCard** - 加载骨架屏
  - 提供良好的加载体验

- **Navigation** - 导航栏
  - 页面快速切换
  - 当前页高亮显示

### 页面功能
- **Feed** - 推荐流页面
  - 垂直滚动浏览论文
  - 无限滚动加载更多
  - 分类筛选
  - 键盘快捷键支持（j/k 上下切换，l 点赞，f 收藏）

- **Search** - 搜索页面
  - 防抖搜索输入
  - 搜索结果展示
  - 空状态和错误处理

- **Favorites** - 收藏列表页面
  - 展示用户收藏的论文
  - 本地存储持久化

### 状态管理
- **AppContext** - 全局状态
  - 论文列表管理
  - 点赞/收藏状态
  - 分类选择
  - 搜索状态

### Hooks
- **usePapers** - 论文数据管理
  - 支持分类筛选
  - 分页加载
  - 错误处理

- **useSearch** - 搜索功能
  - 防抖处理
  - 搜索结果管理

- **useLocalStorage** - 本地存储
  - 点赞列表持久化
  - 收藏列表持久化
  - 浏览历史记录

- **useKeyboard** - 键盘快捷键
  - 自定义快捷键映射
  - 预定义论文浏览快捷键

### 工具函数
- **text.ts** - 文本处理
  - 格式化作者列表
  - 获取分类名称
  - 文本截断

- **date.ts** - 日期处理
  - 相对时间格式化（3小时前）
  - 完整日期格式化

- **storage.ts** - 本地存储
  - 通用存储操作
  - 用户偏好管理

### 样式设计
- 深色主题（抖音风格）
- 响应式设计
- 流畅的过渡动画
- 自定义滚动条样式

### 验证方式
```bash
# 启动前端
cd frontend && npm run dev

# 访问地址
# http://localhost:5173

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

```bash
# .env
VITE_API_BASE_URL=http://localhost:8080
```
